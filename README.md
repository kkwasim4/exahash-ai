# Kkwasim4 Spot Trading Scanner
A real-time crypto spot scanner featuring:
- OBV indicator
- Mini-chart per pair
- Auto-loop price refresh
- Token scanner using Alchemy API
- Clean Coinbase-style UI

uvicorn app:app --reload
